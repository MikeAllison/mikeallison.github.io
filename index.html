About Me

For as long as I can remember, I’ve always had a desire to learn and create. This started with building with Legos as a child, learning to cook in middle school, and finally teaching myself to play multiple instruments in my teens so that I could write complete songs.

Shortly after college, I picked up some books to learn HTML, CSS, and JavaScript so that I could build small websites for bands that I was a part of. Friends took notice of this and asked me to build sites for them. I found this incredibly fun and rewarding because it helped satisfy my desire to learn and create. Unfortunately, I had to put these projects on hold after starting a new job and not having enough time to dedicate to them.

A few years later, I cobbled together another small website for my latest band. Although my skills were pretty rusty, it was still as fun and rewarding as it had been in the past. I immediately started to regret my decision to not pursue web development as a career. I signed up for Treehouse and started re-learning the basics. Since then I’ve taken courses on a variety of topics from Code School, Pluralsight, Launch School, Udemy, and Udacity. Some of these topics include: HTML5, CSS3, JavaScript, jQuery, Ruby on Rails, SASS, Bootstrap, Foundation, and responsive design.

Recently, I’ve learned about using development tools such as Gulp and Grunt, the methodologies behind using front end frameworks like Ember and Backbone, and am currently working through a tutorial on Angular.
